# 🍜 Nom-Nom

Nom-Nom is a chaotic little food app built for fun.

Search recipes, discover new dishes, save your favorites, spin the recipe roulette when you can't decide what to eat, and get distracted by cooking videos you definitely didn't plan to watch.

## ✨ Features

* 🍕 Search and discover recipes
* 🎲 Recipe Roulette for random meal suggestions
* ❤️ Save recipes to your wishlist
* 💬 Interactive food chat
* 🎥 Watch related YouTube Shorts and recipe videos
* 🌶️ Food recommendations with a bit of personality

## 🛠️ Tech Stack

### Frontend

* React
* Vite

### Backend

* Node.js
* Express
* Socket.io

### Database

* MongoDB

## 🚀 Running Locally

Install dependencies:

```bash
npm install
```

Start the frontend:

```bash
npm run dev
```

Start the backend:

```bash
node index.js
```
